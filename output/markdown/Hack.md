### Slide 59

- OVERVIEW

Augment hacks cause one of the three augment slots offered during an augment round to be “special” in some way. See more.

When a hack occurs, the affected augment slot will be given more rerolls (+4). This is to allow the player a little bit more agency when it comes to engaging with the mechanic, offering a little more chance to fish for a hacked augment they like. These bonus rolls would stack on top of any other bonus augment rolls the player might have (for example, if they already had +3 augment rerolls from Beggars can be Choosers, their hacked slot would have 8 rerolls total.)

These hacked augment rounds always affect all players at the same time.

Hacked Augment slots will be the same rarity as the Augment that got replaced. That is, in a Gold Augment round, the hacked slot would still be showing you Gold Augments (the “Rarity Hack” in the Hack Types section is an exception to this rule).
